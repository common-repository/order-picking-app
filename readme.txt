== Description ==

Make your life easier by using the Orderpicking App. You'll never be inefficient if the Orderpicking App is installed in your store. We assist you in all aspects of your webshop. From intelligent selecting to order packing, we have you covered.

Connecting the Orderpicking App to your Woocommerce webshop is simple and quick. Within an hour, you'll be online with the Orderpicking App. You're able to pick and pack your orders three times faster and with greater accuracy.

== Installation ==

1. Install our plugin.
2. Activate our plugin
3. Start your 30 days trial direct inside Woocommerce
4. Install outr app for IOS or Android
5. Scan or enter your API key
6. Yeh, start picking!
